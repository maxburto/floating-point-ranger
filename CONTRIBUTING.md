# Contributing

This GitHub repo is a mirror. Development happens on my self-hosted Gitea, and every push
there force-syncs here - so nothing merged directly on GitHub survives. Issues and pull
requests are still welcome: I read them here, pull the changes into my Gitea repo, and
they mirror back out with credit. It just means a merged PR closes as "closed" rather
than "merged", because the commit arrives via the mirror.

If you're planning something bigger than a small fix, open an issue first so we can talk
about it before you sink time in.
